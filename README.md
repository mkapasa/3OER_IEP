# 3OER_IEP
3ο Δείγμα Πόρου για την προκύρηξη του ΙΕΠ
Πρόκειται για ένα πρωτότυπο εκπαιδευτικό σενάριο που χρησιμοποιεί την προσομοίωση του οπερονίο της λακτόζης στην πλατφόρμα προσομοιώσεων PhET Colorado (https://phet.colorado.edu/en/simulations/gene-machine-lac-operon). Η προσομοίωση απαιτεί τοπικά εγκατεστημένη Java ή ανοίγει διαδικτυακά με τη χρήση cherrpj. To εκπαιδευτικό αυτό σενάριο σχεδιάστηκε προκειμένου οι μαθητές της Γ' Λυκείου του προσανατολισμού υγείας να κατανοήσουν τον τρόπο λειτουργίας του οπερονίου της λακτόζης που αφορά στην παράγραφο με τίτλο "Η γονιδιακή ρύθμιση στους προκαρυωτικούς οργανισμούς του κεφ 2 του σχολικού εγχειριδίου.
